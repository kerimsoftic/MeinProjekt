Protokol                           Kerim Softic 

1. git cmd öffnen
2. mkdir MeinProjekt neuen Ordner erstellen
3. cd MeinProjekt in den Ordner reingehen
4. git init eingeben erstellt neues Git-Repository
5. echo "Das ist mein erstes Git-Projekt." > readme.txt erstellt neue Datei
6. git add readme.txt eingeben
7. git commit -m "Erste Version von readme.txt hinzugefügt" speichert Änderung im Repository
8. auf GitHub neues Repository erstellen
9.git remote add origin https://github.com/deinBenutzername/MeinProjekt.git  lokales Repository mit GitHub verbinden
10.git push -u origin main funktioniert nicht kommt ein fehler raus
